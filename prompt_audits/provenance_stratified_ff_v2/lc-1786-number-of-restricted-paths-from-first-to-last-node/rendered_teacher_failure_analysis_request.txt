## Candidate Analysis

**Candidate 1: Dijkstra from node n + DP on DAG of decreasing distance**
- Compute shortest distance from every node to n using Dijkstra (since weights positive, O((n+m) log n)).
- A restricted path from 1 to n must have strictly decreasing distance-to-n at each step. This defines a DAG: edge u→v exists if dist[u] > dist[v].
- Count number of paths from 1 to n in this DAG via DP (topological order by decreasing dist).  
- Correctness: Strictly decreasing distance ensures no cycles; DP counts all paths.  
- Complexity: Dijkstra O((n+m) log n) ~ 60000 log 20000; DP O(n+m). Well within limits.

**Candidate 2: BFS-like DP on original graph with memoization**
- Without precomputing distances, attempt to define DP[u] = number of restricted paths from u to n, recursively using neighbors where dist[u] > dist[v].  
- But dist values are unknown; computing them on the fly would require solving shortest paths anyway.  
- Rejected: Inefficient and redundant; Dijkstra is necessary for correctness.

## Selected Algorithm

Dijkstra from node n to compute dist[] (shortest distance to n), then DP on DAG sorted by decreasing dist.

## State or Invariant

- `dist[v]` = shortest distance from v to n (Dijkstra invariant: settled nodes have correct distance).
- `dp[v]` = number of restricted paths from v to n.  
- Process nodes in decreasing order of dist (strictly decreasing ensures topological order).  
- Base: `dp[n] = 1`.  
- Transition: For each edge (v, u) with dist[v] > dist[u], `dp[v] = (dp[v] + dp[u]) % MOD`.

## Complexity

- Time: O((n + m) log n) for Dijkstra + O(n + m) for DP = O((n+m) log n).  
- Space: O(n + m) for adjacency list, dist array, dp array.  
- With n ≤ 20000, m ≤ 40000, easily fits 256 MB and 4 seconds.

```python
class Solution:
    def countRestrictedPaths(self, n: int, edges: list[list[int]]) -> int:
        MOD = 1000000007
        adj = [[] for _ in range(n + 1)]
        for u, v, w in edges:
            adj[u].append((v, w))
            adj[v].append((u, w))
        import heapq
        dist = [float('inf')] * (n + 1)
        dist[n] = 0
        pq = [(0, n)]
        while pq:
            d, u = heapq.heappop(pq)
            if d != dist[u]:
                continue
            for v, w in adj[u]:
                nd = d + w
                if nd < dist[v]:
                    dist[v] = nd
                    heapq.heappush(pq, (nd, v))
        order = list(range(1, n + 1))
        order.sort(key=lambda x: dist[x], reverse=True)
        dp = [0] * (n + 1)
        dp[n] = 1
        for u in order:
            if dp[u] == 0:
                continue
            for v, _ in adj[u]:
                if dist[u] > dist[v]:
                    dp[u] = (dp[u] + dp[v]) % MOD
        return dp[1]
```
